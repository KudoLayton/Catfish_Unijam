Detailed Description

Component that contains liquid material parameters.

It doesn't execute anything by itself, it is used by ZibraLiquid instead.

It's separated so you can save and apply presets for this component separately.

◆ FluidMeshMaterial

Material FluidMeshMaterial

Material that will be used to render liquid in Mesh Render mode.

If you want to create your own material, you'll need to use default one as a reference.

This is the material that gets parameters defined in ZibraLiquidMaterialParameters

If you set it to null in Editor, it'll revert to default.

◆ UpscaleMaterial

Material UpscaleMaterial

Material that will be used to upscale liquid in Mesh Render mode.

Most users won't need to customize this material, but if you want to create your own material, you'll need to use default one as a reference.

Has no effect unless you enable downscale in ZibraLiquid component.

If you set it to null in Editor, it'll revert to default.

◆ SDFRenderMaterial

Material SDFRenderMaterial

Material that will be used to upscale liquid in Mesh Render mode.

This material is meant to only be used for debugging.

We don't expect that anyone will need to modify this, but if you want to create your own material, you'll need to use default one as a reference.

Has no effect unless you enable VisualizeSceneSDF in ZibraLiquid component.

If you set it to null in Editor, it'll revert to default.

◆ FoamBlurring

float FoamBlurring = 0.0001f

(Pro version only) Controls how much foam info does particle get from the grid.

Higher values makes particles get foam values closer to the avarage value. Which blurs foam values. Also makes foam dissapear as a result.
